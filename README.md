정확한 시간 추적\n- 🎮 **레트로 게임 컨셉** - 8비트 게임 스타일 UI\n- 🏆 **실시간 랭킹 시스템** - Firebase 기반 멀티유저 랭킹\n- 🎉 **축하 이펙트** - 9단계 시간별 특별 애니메이션\n- 🧠 **AI 활동 추천** - 시간대별 맞춤 생산성 활동 제안\n- 💬 **50가지 비난 메시지** - 랭킹/시간대 기반 스마트 메시지\n- 📱 **완전 반응형** - 모든 디바이스 지원\n- 🔥 **극한 모드** - 5분 후 활성화되는 특별 모드\n\n## 🚀 라이브 데모\n\n**👉 [https://pg-parunson.github.io/timewaster/](https://pg-parunson.github.io/timewaster/)**\n\n## 🎨 스크린샷\n\n### 메인 화면\n- 레트로 게임 스타일 타이머\n- 실시간 랭킹 보드\n- 스마트 활동 추천\n\n### 축하 이펙트\n- 2분, 5분, 10분... 단계별 특별 애니메이션\n- 파티클 효과와 배경 플래시\n\n## 🛠️ 기술 스택\n\n### Frontend\n- **React 18.2.0** - 메인 프레임워크\n- **TailwindCSS 3.3.3** - 스타일링\n- **Lucide React** - 아이콘\n- **Vite 4.4.5** - 빌드 도구\n\n### Backend & Database\n- **Firebase Realtime Database** - 실시간 랭킹 시스템\n- **Firebase Hosting** - 배포 (자동 배포)\n\n### 기타\n- **GitHub Actions** - CI/CD\n- **쿠팡 파트너스** - 수익화\n- **Google Analytics** - 분석\n\n## 📁 프로젝트 구조\n\n```\ntimewaster-project/\n├── src/\n│   ├── components/          # React 컴포넌트들\n│   │   ├── SiteHeader.jsx   # 헤더 + 시간 표시\n│   │   ├── MessageSection.jsx # 비난 메시지\n│   │   ├── RankingSection.jsx # 실시간 랭킹\n│   │   ├── CelebrationEffect.jsx # 축하 이펙트\n│   │   └── ...\n│   ├── hooks/               # 커스텀 훅들\n│   │   ├── useTimerLogic.js # 메인 타이머 로직\n│   │   ├── useModalLogic.js # 모달 관련 로직\n│   │   └── useCelebrationSystem.js\n│   ├── data/                # 데이터 및 메시지\n│   │   ├── roastMessages.js # 50가지 비난 메시지\n│   │   ├── timeBasedActivities.js # 활동 추천\n│   │   └── celebrationEffects.js\n│   ├── services/            # 외부 서비스\n│   │   ├── rankingService.js # Firebase 랭킹\n│   │   └── liveFeedService.js # 실시간 피드\n│   └── utils/               # 유틸리티\n├── docs/                    # 문서들\n│   ├── development/         # 개발 관련\n│   ├── features/           # 기능 설명\n│   ├── troubleshooting/    # 문제 해결\n│   └── archive/            # 아카이브\n├── public/                 # 정적 파일들\n└── README.md              # 이 파일\n```\n\n## 🎮 주요 기능 상세\n\n### 1. 실시간 랭킹 시스템\n- Firebase Realtime Database 연동\n- 익명 닉네임 자동 할당 (\"시간여행자\", \"우주탐험가\" 등)\n- TOP 10 실시간 업데이트\n- 일간/주간/월간/전체 랭킹 지원\n\n### 2. 스마트 메시지 시스템\n- **50가지 비난 메시지** (기본/중급/고급 단계별)\n- **랭킹 기반 특별 메시지** (1위, 상위권, 톱10 등)\n- **시간대별 메시지** (새벽/오전/오후/저녁 각 5개)\n- **확률 기반 선택** (랭킹 20% + 시간대 15% + 일반 65%)\n\n### 3. AI 활동 추천 시스템\n- **뇌과학 기반 시간대 최적화** (7개 시간대별 분석)\n- **200+ 실행 가능한 활동** (난이도별 분류)\n- **개인화된 추천** (사용자 패턴 학습)\n- **상황 인식 제안** (에너지, 집중도, 시간대 고려)\n\n### 4. 축하 이펙트 시스템\n- **9단계 시간별 이펙트** (2분~2시간)\n- **다양한 애니메이션** (bounce, pulse, float, rainbow 등)\n- **파티클 효과** (이모지 떠다니기)\n- **배경 플래시** (반짝거리는 화면)\n\n## 🔧 개발 환경 설정\n\n### 1. 저장소 클론\n```bash\ngit clone https://github.com/pg-Parunson/timewaster.git\ncd timewaster\n```\n\n### 2. 의존성 설치\n```bash\nnpm install\n```\n\n### 3. 개발 서버 실행\n```bash\nnpm run dev\n```\n\n### 4. 빌드\n```bash\nnpm run build\n```\n\n### 5. 배포\n```bash\nnpm run deploy\n```\n\n## 🔥 최신 업데이트 (v3.0)\n\n### 🎨 **UI 개선 완료** (2025-05-25)\n- ✅ **시간 표시를 타이틀 옆으로 이동** - 더 세련된 배치\n- ✅ **뱃지 색상 밝고 선명하게 개선** - emerald, cyan 등\n- ✅ **5분 축하 이펙트 안정화** - UI 깨짐 방지\n- ✅ **랭킹 보드 레이아웃 1:2 비율로 확대**\n- ✅ **코드 리팩토링** - App.jsx 28KB → 16KB로 축소\n\n### 🎯 **v3.0 혁신적 기능**\n- ✅ **시간대별 세밀한 활동 매칭** (1분~60분+ 10단계)\n- ✅ **뇌과학 기반 시간대 최적화** (7개 시간대별 분석)\n- ✅ **비난 메시지 50개 확장** + 랭킹/시간대 기반\n- ✅ **Firebase 랭킹 시스템** 완전 구현\n\n## 📊 성능 지표\n\n- **번들 크기**: ~500KB (압축 후)\n- **초기 로딩**: <2초\n- **실시간 업데이트**: 30초 간격\n- **Firebase 연결**: <1초\n- **SEO 점수**: 95/100\n\n## 🤝 기여 방법\n\n1.
