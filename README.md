# Timewaster Calculator 🕒

[![Deploy Status](https://img.shields.io/badge/deploy-ready-brightgreen)](https://pg-parunson.github.io/timewaster)
[![React](https://img.shields.io/badge/React-18.2.0-blue)](https://reactjs.org/)
[![TailwindCSS](https://img.shields.io/badge/TailwindCSS-3.3.3-blue)](https://tailwindcss.com/)
[![Version](https://img.shields.io/badge/version-v3.0-orange)](https://github.com/pg-Parunson/timewaster)

> **당신이 낭비한 시간을 실시간으로 계산하고, 더 나은 활용법을 제안하는 혁신적인 웹사이트**

## 🎯 프로젝트 개요

단순한 "시간 낭비 계산기"에서 **개인 맞춤형 생산성 코치**로 진화한 웹사이트입니다. 사용자가 사이트에서 머무는 시간을 측정하며, 유쾌한 비난조 멘트와 함께 구체적이고 실행 가능한 대안 활동을 제안합니다.

## ✨ v3.0 혁신적 기능 (최신)

### 🧠 **시간대별 세밀한 활동 매칭**
- **1분~60분+ 10단계 구간**: 각 시간에 맞는 구체적 활동 추천
- **뇌과학 기반 시간대 최적화**: 7개 시간대별 뇌상태 분석 (새벽 창의성~저녁 성찰)
- **200+ 실행 가능한 활동**: 난이도별 분류된 실제 실행 가능한 활동들

### 🤖 **스마트 메시지 시스템**
- **확률 기반 지능형 선택**: 랭킹 20% + 스마트 40% + 시간대 20% + 일반 20%
- **개인화된 AI 추천**: 사용자 패턴 학습 및 맞춤형 제안
- **상황 인식**: 에너지 레벨, 집중도, 시간대를 고려한 맥락적 제안

### 📱 **인터랙티브 활동 추천 UI**
- **대화형 활동 선택 카드**: 클릭 가능한 아름다운 추천 인터페이스
- **실시간 상태 표시**: 뇌 상태, 에너지, 집중도 시각화
- **실행 유도 버튼**: "🚀 지금 바로 시작하기"로 실제 행동 연결

## 🏆 v2.0 고급 기능

### 🔥 **Firebase 실시간 랭킹**
- **실시간 사용자 랭킹**: TOP 10 + 현재 사용자 하이라이트

- **랭킹 기반 특별 메시지**: 1위, 상위권, 첫 랭킹 등 상황별 메시지

### 🎨 **완전 리뉴얼된 UI/UX**
- **50개 스마트 비난 메시지**: 기본/중급/고급 + 랭킹/시간대 기반
- **축하 이펙트 시스템**: 9단계 마일스톤 달성 애니메이션
- **세련된 모달 시스템**: 모든 alert 대신 현대적 모달 UI

## 🎮 핵심 기능

## 🚀 빠른 시작

### 필수 요구사항
- Node.js 16+ 
- npm 또는 yarn

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/pg-Parunson/timewaster.git
cd timewaster

# 의존성 설치
npm install

# 개발 서버 시작
npm run dev

# 프로덕션 빌드
npm run build

# GitHub Pages 배포
npm run deploy
```
