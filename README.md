## 🎯 **핵심 기능**

### **⏰ 실시간 시간 측정**
- 대형 레트로 타이머 디스플레이
- 번쩍거리는 위기 모드 (5분+ 경과시)
- 레트로 타이핑 효과와 함께하는 실시간 메시지

### **🎭 50개 지능형 비난 시스템**
- 시간대별 맞춤 메시지 (새벽/오전/오후/저녁)
- 랭킹 기반 특별 메시지 (1위/상위권/신규 진입)
- 뇌과학 기반 생산적 활동 추천

### **🏆 실시간 Firebase 랭킹**
- 실시간 TOP 10 랭킹 시스템
- 30개 랜덤 익명 닉네임 자동 할당
- 명예의 전당 스타일 UI

### **🎊 화려한 축하 이펙트**
- 2분: 파란색 테마 + 파티클 애니메이션
- 5분: 초록색 테마 + 화려한 축하 효과
- 배경 플래시와 완벽 동기화

### **📊 실시간 통계 시스템**
- 실제 동시 접속자 수 표시
- 전체 사이트 방문 횟수 누적
- 총 시간낭비 통계

### **🛍️ 스마트 광고 시스템**
- 쿠팡 파트너스 연동
- 시간대별 맞춤 상품 추천
- 레트로 게임 스타일 광고 UI

### **📱 완벽한 SNS 공유**
- 카카오톡 리치 카드 지원
- X(트위터) 공유
- 링크 복사 기능

## 🚀 **빠른 시작**

### **설치 및 실행**
```bash
# 저장소 클론
git clone https://github.com/pg-Parunson/timewaster.git
cd timewaster

# 의존성 설치 (Galmuri 폰트 포함)
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# GitHub Pages 배포
npm run deploy
```

## 🎯 **사용자 경험**

### **✨ 완벽한 여정**
1. **0초**: 깔끔한 레트로 UI에 즉시 몰입

- **실시간 랭킹**: 다른 사용자와 경쟁
- **레벨업 시스템**: 시간에 따른 단계별 도전
- **업적 시스템**: 다양한 마일스톤 달성
- **소셜 기능**: SNS 공유로 친구들과 경쟁

## 🛠️ **개발 가이드**

### **프로젝트 구조**
```
src/
├── components/     # UI 컴포넌트들
├── hooks/         # 커스텀 훅스
├── services/      # Firebase 서비스
├── data/          # 정적 데이터
├── utils/         # 유틸리티 함수들
└── App.jsx        # 메인 애플리케이션
```

### **핵심 컴포넌트**
- `TimerSection`: 대형 레트로 타이머
- `MessageSection`: 지능형 메시지 시스템  
- `RankingSection`: 실시간 랭킹 (명예의 전당)
- `AdSection`: 스마트 광고 시스템
- `CelebrationEffect`: 축하 이펙트 애니메이션
